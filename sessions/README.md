# Sessions

本目录用于存放开发过程中的 session 记录，便于回溯实验过程、设计取舍与问题定位。

## 命名约定

- 建议使用 `SESSION-YYYY-MM-DD.md` 格式（按日期递增）。
- 单日多次记录可在文件名后追加短后缀，例如：`SESSION-2026-04-12-a.md`。
