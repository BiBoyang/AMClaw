# AGENTS.md

@scope:src/command_router:v1

## 模块定位

命令路由层，负责把微信文本解析成统一的路由意图。

## 当前职责

1. 解析聊天流控制信号：`..`、`!!`、普通文本。
2. 解析命令型查询：`状态 <task_id>`、`最近任务`、`重试 <task_id>`。
3. 识别链接提交消息并输出统一的路由动作对象。

## 后续职责

1. 解析 `今日整理` 等更多命令。
2. 做 URL 提取、规范化与基础校验增强。
3. 统一承接聊天流、链接流与只读查询命令入口。

## 不做事项

1. 不直接调用 iLink 接口。
2. 不直接写数据库。
3. 不管理会话定时器。
4. 不执行抓取或分类。
