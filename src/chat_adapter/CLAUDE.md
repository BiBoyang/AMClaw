# CLAUDE.md

@scope:src/chat_adapter:v1

## 模块定位

当前微信能力实现目录，负责 iLink 登录、长轮询、消息接入、会话提交与发送回复。  
内容应与同目录 `AGENTS.md` 保持同步；若冲突，以 `AGENTS.md` 为准。

## 当前边界

1. 维护与 iLink 协议对接逻辑。
2. 保障接收、入站落库、会话提交与回复链路可运行。
3. 负责消息去重、`context_token` 缓存与会话层接线。
4. 根据会话超时动态调整轮询等待时间，避免提交时机被长轮询阻塞。
5. 在入口处分流聊天流、链接流、只读任务命令与人工补正文命令。
6. 支持待人工补录任务的查询与恢复命令接线。

## 演进约束

1. 新增逻辑优先保持与现有接口兼容。
2. 协议字段调整必须附带调试日志。
3. 聊天接入层只负责协议与主链路接线，不承载文章抓取、分类、归档逻辑。
4. 中长期会向 `src/chat_gateway/` 结构化迁移。
