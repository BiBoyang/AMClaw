@scope:src/agent_core:v1

## 模块定位

Agent 执行运行时，负责 LLM 调用、计划表达、工具调度、observation 承接、状态推进与最小失败处理。

## 预期职责

1. 接收上游运行上下文并调用模型。
2. 维护最小多步 `ReAct` / `Plan-aware` 执行循环。
3. 根据策略触发允许的工具链并承接 `observation`。
4. 推进 `step_status`、完成条件、失败语义与最小 `replan`。

## 不做事项

1. 不直接操作微信接口。
2. 不直接决定文件写入路径。
3. 不绕过模式策略执行高风险动作。
4. 不在本模块直接承担浏览器抓取、微信协议与持久化细节。
