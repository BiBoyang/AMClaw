# CLAUDE.md

@scope:src/pipeline:v1

## 模块定位

处理流水线层，负责任务消费与“快照 -> 抽取 -> 分类 -> 归档”。  
内容应与同目录 `AGENTS.md` 保持同步；若冲突，以 `AGENTS.md` 为准。

## 当前职责

1. 消费 `pending` 任务。
2. 根据域名选择 HTTP 或浏览器抓取链路。
3. 生成原始 HTML、截图和 Markdown 归档产物。
4. 对公众号页面识别验证码页、权限页、错误页，并支持人工补录接续归档。

## 后续职责

1. 按状态机推进任务阶段。
2. 统一处理每个阶段的输入输出结构。
3. 失败时可定位到具体阶段并支持重试。

## 不做事项

1. 不解析微信命令。
2. 不管理账号登录会话。
3. 不定义模式权限策略。
4. 不负责决定任务何时被查询或重试。
